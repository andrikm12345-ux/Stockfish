# Stockfish Chess Bot

Бот для автоматической игры на Lichess и Chess.com. Использует движок Stockfish + Playwright для управления браузером. Имитирует живого человека: задержки, ошибки, дебютная книга, иногда проигрывает по времени.

---

## Быстрый старт

**1. Скачать bot.js**
```
curl -L -o bot.js https://raw.githubusercontent.com/andrikm12345-ux/Stockfish/main/bot/bot.js
```

**2. Установить зависимости** (один раз)
```
npm install chess.js playwright
npx playwright install chromium
```

**3. Положить рядом `stockfish.exe`** (скачать отдельно)

**4. Запустить**
```
node bot.js
```

## Настройки при запуске

| Переменная | По умолчанию | Описание |
|---|---|---|
| `DEPTH=12` | 18 | Глубина поиска движка |
| `SITE=chess` | lichess | Сайт: `lichess` или `chess` |

Пример:
```
DEPTH=8 SITE=lichess node bot.js
```

| Команда | Пример | Описание |

## Авто-глубина (работает по умолчанию)

При старте каждой новой игры бот читает часы и сам выставляет глубину:

| Контроль | Время | Глубина |
|---|---|---|
| Пуля | < 3 мин | `depth 5` |
| Блиц | 3–10 мин | `depth 8` |
| Рапид | 10+ мин | `depth 12` |

Команда `d N` отключает авто. Команда `a` включает обратно.

---

## Скилл — сколько ошибок делает бот

```
s 0   → играет как новичок, зевает постоянно
s 13  → иногда ошибается, ~85-90% точность — хорошо для игры с людьми
s 15  → редкие ошибки, ~92% точность
s 20  → почти идеально, палевно против людей
```

**Рекомендации:**
- Против людей (пуля/блиц): `s 13–15`
- Против Stockfish / тренировка: `s 20`

## Как работает бот (логика)

### 1. Дебютная книга
- 37 дебютных линий (Испанская, Сицилианская, Итальянская, Французская, Каро-Канн, Ферзевый гамбит, Индийские, Лондонская, Рети и другие)
- В дебюте ходы играются быстро: 200–800 мс
- После 3-го хода есть **15% шанс** отклониться от книги и играть движком — имитирует человека, который не помнит теорию до конца

### 2. Движок (Stockfish)
- MultiPV = 3 (считает топ-3 хода)
- **20% шанс** сыграть 2-й лучший ход (если разница < 80 очков)
- **6% шанс** сыграть 3-й лучший ход (только в равной позиции)
- В выигранной позиции (> +200) и проигранной (< -100) — всегда лучший ход

### 3. Задержки (имитация человека)
- Цейтнот < 5 сек: 80–200 мс
- Цейтнот < 10 сек: 130–300 мс
- Обычная игра: 2–5% от оставшегося времени
- Иногда думает дольше (18% шанс — увеличение в 1.3–2.3x)
- Серии быстрых ходов (18% шанс — 2–5 ходов подряд быстро)

### 4. Флагование в пуле
- При остатке < 8 секунд, в ненапряжённой позиции — **7% шанс** намеренно просрочить время
- Имитирует реального человека в цейтноте

### 5. Движение мыши
- 60% ходов: сначала наводится на соседнюю клетку, потом на нужную
- Добавляется случайный сдвиг и задержка между кликами

## Режимы подключения браузера

**Авто (по умолчанию):** бот сам запускает Chrome с отдельным профилем.

**CDP режим** (твой собственный Chrome):
1. Запусти Chrome с флагом:
```
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222
```
2. Запусти бота:
```
CDP=1 node bot.js
```
